docsディレクトリは、以下のようなディレクトリが含まれています。


txtディレクトリ

UDFVについてのテキスト形式のドキュメントを収録しています。
以下の４つのファイルから構成されています。

udfv.txt         - UDFVについての概略
requirement.txt  - 必要動作環境とその構築方法について
udfvtools.txt    - UDFV toolsについての解説
xml.txt          - XMLドキュメントファイルの解説
frontend.txt     - UDFV toolsの1つ、Frontendの解説
udfvgui.txt      - UDFV GUIについての解説

htmlディレクトリ

txtディレクトリにあるテキスト形式のドキュメントを
HTML化したものです。ブラウザでindex.htmlを開いてください。
内容はテキストドキュメントと同じです。
