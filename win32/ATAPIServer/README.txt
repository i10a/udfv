■準備

コンパイルするには以下のツールを使用する。

	+ Visual C++ 6.0
	+ DDK

	※ DDKはサンプルも含めて全て展開する。


また、DDK をインストールした後以下の作業を実行する。

	1. Visual C++ のメニューバーから「ツール」−「オプション」を選び、オプ
	   ションダイアログを起動。

	2. 「ディレクトリ」タブを選択し、「表示するディレクトリ」を「インクルード
	   ファイル」にする。

	3. "C:\WINDDK\2600.1106\inc\crt", "C:\WINDDK\2600.1106\inc\ddk\wxp",
	   "C:\WINDDK\2600.1106\inc\wxp" を追加する。(OS にあわせて変更する)

	4. 「3.」で追加したディレクトリを矢印アイコンで一番上に持ってくる。

	5. 「OK」をクリック。

	6. Visual C は暴走すると設定内容を忘れることがあるので、念のため Visual C
	   を終了する。


■コンパイル

Visual C++ を起動し、"UDFV\trunk\win32\ATAPIServer" ディレクトリの
"ATAPIServer.dsp" を読み込み、コンパイルする。


■実行

F5 キーで起動する。


■Java

Windows 側を起動したら "UDFV\trunk" にカレントを移動し、"Test\UDFVGui.bat" で
起動する。

アイコンの左から３番目、あるいはメニューバーの「File」−「Read from Disc」を
選択する。

「IP Address」に、Windows の IP アドレス、「Drive」にディスクを入れたドライブ
レターを入力し、「OK」ボタンをクリック。(毎回入力するのが面倒な場合は「ツール」
−「オプション」−「IP」でデフォルト値を変更することができる)

以上。
